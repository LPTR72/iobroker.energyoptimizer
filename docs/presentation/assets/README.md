# Presentation assets

This folder contains reusable visual assets for the public presentation pages.

Official project logos are not maintained here. The single source of truth for logos is [`../../images/`](../../images/), especially `Logo_Large.png` and the square icon files.

## Files

| File | Purpose |
| --- | --- |
| [`architecture.svg`](architecture.svg) | Public architecture overview. |
| [`pipeline.svg`](pipeline.svg) | Optimization pipeline overview. |
| [`timeline.svg`](timeline.svg) | Project maturity and roadmap timeline. |
| [`history.svg`](history.svg) | History Service concept overview. |
| [`virtual-assets.svg`](virtual-assets.svg) | Pattern-based Virtual Energy Assets concept overview. |
| [`simulation.svg`](simulation.svg) | Current Simulation Runtime and planned Simulation Framework overview. |
| [`future-step.svg`](future-step.svg) | Later-stage recommendation path placeholder. |

## Style

- SVG format for GitHub and GitHub Pages compatibility.
- Light background.
- Green for implemented foundations.
- Orange for planned or active work.
- Gray for future or dormant areas.
- Minimal text so diagrams remain readable in Markdown.
