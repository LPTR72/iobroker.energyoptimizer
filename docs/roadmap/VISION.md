# Vision

ioBroker Energy Optimizer aims to become a transparent, vendor-neutral energy optimization platform for ioBroker.

The project builds from safe read-only data handling toward explainable recommendations and later separately approved integration milestones.

## Core principles

- Model physical energy behavior instead of vendor APIs.
- Keep domain logic deterministic and testable.
- Use historical context as reusable knowledge.
- Treat detected patterns as hypotheses until confirmed by the user.
- Keep recommendations separate from later device behavior.
- Preserve public-state compatibility unless a migration is explicitly approved.
