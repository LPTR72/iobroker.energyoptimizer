# Project status

## Current state

The project is under active development.

The public runtime currently reads configured source states, mirrors live values, calculates fixed-tariff import costs, publishes diagnostics, and exposes read-only recommendation data.

## Implemented foundations

- Runtime polling and live mirroring
- Fixed-tariff cost calculation
- Generic energy assets and normalized configuration
- Analysis, forecast abstraction, prediction, evaluation, and recommendation foundations
- Read-only simulation runtime
- Structured recommendation projection
- Dormant planning-domain semantics
- History Service domain foundation, pending architecture review closure

## Planned areas

- History Service runtime integration
- SQL-backed historical data repository direction
- Forecast, tariff, and weather providers
- Pattern Recognition Engine
- Pattern-based Virtual Energy Assets
- Simulation Framework
- Later integration milestones
