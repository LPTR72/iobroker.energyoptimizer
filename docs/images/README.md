Current files:

- `logo-icon.png` — reusable icon placeholder for future adapter, social-preview, and documentation branding

Future planned files:

- `architecture.svg` — layer and boundary overview
- `pipeline.svg` — domain pipeline
- `execution.svg` — recommendation, planning, capability, and execution flow
- `forecast.svg` — forecast-provider and prediction flow

Prefer editable SVG sources, accessible labels, and references that remain readable in both light and dark GitHub themes.
